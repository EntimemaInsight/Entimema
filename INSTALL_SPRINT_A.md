# Install — Sprint A Premium Identity

1. Close the local development server.
2. Copy the contents of the `entimema` folder over your current project folder.
3. Do not replace your local `.git` folder (it is not included in this release).
4. Run:

```powershell
cd C:\Users\user\entimema
npm install
npm run dev
```

5. Check `/`, `/services`, `/about`, and `/contact`.
6. Publish only after local approval:

```powershell
git add .
git commit -m "Introduce Sprint A premium identity"
git push origin main
```
