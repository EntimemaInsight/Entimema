# 03 Typography

Typography is restrained and highly legible. Display text is bold but not inflated. Body copy is regular with generous line height. Navigation and labels use medium or semibold weights.

All production values are defined in `styles/tokens.css`.


## v1.0.1 — Unified technical typography
- Единна фамилия: Segoe UI Variable / Segoe UI / Arial.
- Hero използва същия конструктивен характер като заглавията вдясно.
- Hero: 700 weight, line-height 1.105, letter-spacing -0.035em.
- Няма отделен display шрифт. Йерархията се създава само чрез размер и тегло.
