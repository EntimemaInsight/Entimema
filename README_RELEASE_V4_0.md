# ENTIMEMA Design Language 2026 — v4.0

## Scope
This release replaces generic iconography with a bespoke Entimema illustration language and applies a unified premium typography/card system across Services, Approach and Process.

## Key changes
- Six custom service diagrams: finance architecture, risk decisioning, data/AI flow, transformation, CFO dashboard and AI agents.
- Five compact process diagrams with a shared geometric grammar.
- Product-style service cards with diagram-first hierarchy and quieter visual surfaces.
- Process steps rebuilt as compact panels instead of oversized circular icons.
- Consistent stroke, radius, spacing, color and motion rules.
- Refined wordmark scale and type rendering.
- Responsive behavior and reduced-motion support preserved.

## Validation
Run:

```bash
npm install
npm run build
npm run dev
```
