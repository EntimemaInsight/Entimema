# UI Acceptance Criteria

Home is not complete until:
- the complete desktop hero is visible in the reference viewport;
- the logo is crisp at 100%, 125% and 150% OS scaling;
- the wordmark is live text;
- no section looks like a raster image;
- the header is compact and visually subordinate to the hero;
- all five expertise rows are visible and balanced;
- dot density supports whitespace instead of competing with content;
- no horizontal overflow exists at supported breakpoints;
- keyboard focus is visible;
- reduced-motion behavior is supported;
- production build, lint and accessibility checks pass.
