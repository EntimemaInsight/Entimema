# ENTIMEMA v3.3 — Premium icons and visual alignment

## Changed
- Premium 24px SVG icon system with consistent geometry and stroke weight.
- More restrained icon tiles in both service cards.
- Simplified credit-risk network visual.
- Equalized graphic sizes and vertical alignment between the two cards.
- Refined CTA button geometry.

## Install
Copy the contents of this folder over your current Entimema project, then run:

```powershell
npm install
npm run dev
```

After local approval:

```powershell
git add .
git commit -m "Refine premium icons and service-card alignment"
git push origin main
```

## Validation
`npm run build` completed successfully.
